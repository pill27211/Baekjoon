
# [Gold IV] Networking - 3803

[문제 링크](https://www.acmicpc.net/problem/3803)

### 문제 요약

<p> 그래프가 주어졌을 때, 그 그래프의 최소 스패닝 트리를 구해보자. </p>

### 제한

TL : 1sec, ML : 128 MB

1 ≤ V ≤ 50

1 ≤ c ≤ 100

### 성능 요약

메모리: 2160 KB, 시간: 4 ms

### 분류

그래프 이론(grpahs), 최소 스패닝 트리(mst)

### comment

테스트 케이스 분류가 추가된 기본 mst 문제이다.
