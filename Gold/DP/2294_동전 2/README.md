
# [Gold V] 동전 2 - 2294

[문제 링크](https://www.acmicpc.net/problem/2294)

### 문제 요약

<p> 최솟값 동전 DP. </p>

### 제한

TL : 1sec, ML : 128 MB

1 ≤ n ≤ 100

1 ≤ k ≤ 10,000

1 ≤ n_i ≤ 100,000

### 성능 요약

메모리: 2060 KB, 시간: 4 ms

### 분류

다이나믹 프로그래밍(dp)

### comment

dp[i] : i원을 맞추기 위해 필요한 최소한의 동전 개수.

기본 min동전 DP문제이다. dp[k]가 특정됐다면 출력, 불가능하다면 -1 출력.
