
# [Gold IV] 네트워크 연결 - 1922

[문제 링크](https://www.acmicpc.net/problem/1922)

### 문제 요약

<p> 그래프가 주어졌을 때, 그 그래프의 최소 스패닝 트리를 구해보자. </p>

### 제한

TL : 2sec, ML : 256 MB

1 ≤ N ≤ 1,000

1 ≤ M ≤ 100,000

1 ≤ c ≤ 10,000

### 성능 요약

메모리: 3200 KB, 시간: 32 ms

### 분류

그래프 이론(grpahs), 최소 스패닝 트리(mst)

### comment

주어지는 그래프로 mst를 구하면 된다.

이하 설명 생략.
